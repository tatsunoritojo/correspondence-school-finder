# 通信制高校診断：A4レポート出力機能 実装指示書

**作成日**: 2025年12月12日  
**対象プロジェクト**: correspondence-school-finder  
**実装目的**: 既存の画像保存機能を削除し、A4サイズのレポート印刷機能に置き換える

---

## 1. 背景と目的

### 現状の課題
- 現在、診断結果を画像（PNG）として保存する機能が実装されている（`src/components/SaveButton.tsx`を使用）
- 画像形式では情報の再利用性が低く、印刷時のレイアウト調整が困難

### 改善目標
- 診断結果をA4サイズ1ページに収めた専用レポートレイアウトを作成
- ブラウザの印刷機能を活用し、ユーザーが「PDF保存」または「紙への印刷」を選択可能にする
- カラー印刷を前提としつつ、白黒印刷時も視認性を確保

---

## 2. 要件定義

### 機能要件

#### 2.1 レポートの構成
A4サイズ（210mm × 297mm）1ページに以下の内容を含める：

**【ヘッダー部分】**
- レポートタイトル: 「通信制高校診断結果レポート」
- 診断日時: YYYY年MM月DD日形式
- 回答者区分: 「生徒」または「保護者」
- 回答者氏名: レポート生成時に入力

**【セクション1: 診断結果の総合】**
- 左側: タイトルブロック（リッチデザイン）
  - 回答者名の表示（大きめフォント）
  - 回答者区分
  - 診断日
  - 装飾要素（枠線、背景色など）
- 右側: レーダーチャート
  - 8軸のスコアを可視化
  - サイズ: 幅約400-500px（印刷時に適切なサイズに調整）
- レイアウト: 左右50%ずつの横並び配置

**【セクション2: 診断結果の詳細】**
- 8軸の説明カード
  - 横2列×縦4行のグリッドレイアウト
  - 各軸名、スコア、簡潔な説明文

**【セクション3: オープンスクールで聞くべき質問リスト】**
- スコアに基づいた質問項目（最大5項目程度）
- チェックボックス形式で表示

**【フッター部分】**
- サイト名またはロゴ
- 免責事項（必要に応じて）

#### 2.2 デザイン要件

**カラー印刷時:**
- レーダーチャートは現行のカラーリング（Recharts標準色）を使用
- セクションヘッダーは背景色（グレー系: #E5E7EB 等）で視覚的に区別
- 軸の説明ボックスは枠線とアイコンで強調

**白黒印刷時の配慮:**
- レーダーチャートのエリアは薄いグレー（#D1D5DB程度）で塗りつぶし
- 軸線は太線（2px）で明確に表示
- テキストは十分なコントラスト（黒または濃いグレー）
- 重要な情報は太字（font-weight: 600以上）で強調
- ボックスの境界線を1.5px以上の太さで表示

#### 2.3 操作フロー

1. ユーザーが診断を完了し、結果ページ（`ResultPage.tsx`）を表示
2. 「レポートを印刷」ボタンをクリック
3. **名前入力ダイアログが表示される**
   - 回答者の氏名を入力（必須）
   - 「印刷する」ボタンと「キャンセル」ボタン
4. 名前を入力後、「印刷する」をクリック
5. 印刷プレビュー画面が開く（A4レポートレイアウトが表示される）
6. ユーザーが「印刷」または「PDFとして保存」を選択
7. レポートが出力される

## 3. 実装方針

### 3.1 技術スタック
- **フレームワーク**: 既存のReact + TypeScript + Vite環境を維持
- **スタイリング**: Tailwind CSS + カスタムCSS（`@media print`）
- **チャート**: 既存のRechartsライブラリを活用
- **印刷機能**: ブラウザ標準の`window.print()`を使用

### 3.2 アーキテクチャ
- **新規コンポーネント作成**: `src/components/PrintableReport.tsx`
  - 印刷専用のレイアウトを担当
  - 診断結果データをpropsで受け取り、A4サイズに最適化して表示
- **既存コンポーネント削除**: `src/components/SaveButton.tsx`
  - html2canvasを使用した画像保存機能は不要
- **既存コンポーネント修正**: `src/pages/ResultPage.tsx`
  - SaveButtonを削除
  - 新しい「レポートを印刷」ボタンを追加
  - ボタンクリック時に印刷プレビューを表示

**変更後（名前入力ダイアログと印刷ボタンを追加）:**
```tsx
import { useRef, useState } from 'react';
import PrintableReport from '../components/PrintableReport';

export default function ResultPage() {
  const printableRef = useRef<HTMLDivElement>(null);
  const [showNameDialog, setShowNameDialog] = useState(false);
  const [respondentName, setRespondentName] = useState('');

  // ... 診断結果の計算ロジック

  const handlePrintClick = () => {
    // 名前入力ダイアログを表示
    setShowNameDialog(true);
  };

  const handlePrintConfirm = () => {
    // 名前が未入力の場合はエラー
    if (!respondentName.trim()) {
      alert('お名前を入力してください');
      return;
    }

    // ダイアログを閉じる
    setShowNameDialog(false);
    
    // PrintableReportコンポーネントを一時的に表示
    if (printableRef.current) {
      printableRef.current.style.display = 'block';
    }
    
    // 印刷ダイアログを開く
    setTimeout(() => {
      window.print();
      
      // 印刷ダイアログが閉じた後、非表示に戻す
      setTimeout(() => {
        if (printableRef.current) {
          printableRef.current.style.display = 'none';
        }
      }, 100);
    }, 100);
  };

  const handlePrintCancel = () => {
    setShowNameDialog(false);
    setRespondentName('');
  };

  return (
    <div>
      <h1>診断結果</h1>
      <RadarChart data={scores} />
      {/* その他の表示 */}
      
      {/* 印刷ボタン */}
      <button
        onClick={handlePrintClick}
        className="bg-blue-600 text-white px-6 py-3 rounded-lg hover:bg-blue-700"
      >
        レポートを印刷
      </button>

      {/* 名前入力ダイアログ */}
      {showNameDialog && (
        <div className="fixed inset-0 bg-black bg-opacity-50 flex items-center justify-center z-50">
          <div className="bg-white rounded-lg p-6 max-w-md w-full">
            <h2 className="text-xl font-bold mb-4">レポートに表示するお名前</h2>
            <p className="text-sm text-gray-600 mb-4">
              レポートに表示するお名前を入力してください
            </p>
            <input
              type="text"
              value={respondentName}
              onChange={(e) => setRespondentName(e.target.value)}
              placeholder="例: 山田 太郎"
              className="w-full px-4 py-2 border border-gray-300 rounded-lg mb-4"
              autoFocus
            />
            <div className="flex gap-3">
              <button
                onClick={handlePrintCancel}
                className="flex-1 px-4 py-2 border border-gray-300 rounded-lg hover:bg-gray-50"
              >
                キャンセル
              </button>
              <button
                onClick={handlePrintConfirm}
                className="flex-1 px-4 py-2 bg-blue-600 text-white rounded-lg hover:bg-blue-700"
              >
                印刷する
              </button>
            </div>
          </div>
        </div>
      )}

      {/* 印刷用レポート（通常は非表示） */}
      <div ref={printableRef} style={{ display: 'none' }}>
        <PrintableReport
          answers={answers}
          scores={scores}
          respondentType={respondentType}
          respondentName={respondentName}
          diagnosisDate={new Date()}
        />
      </div>
    </div>
  );
}
```

#### 注意点
- `printableRef` を使用してPrintableReportの表示/非表示を制御
- 名前入力ダイアログは `useState` で管理
- 名前が未入力の場合は印刷を実行しない
- ダイアログは固定配置（`fixed inset-0`）でオーバーレイ表示
- `window.print()` 実行前に一時的に表示、終了後に非表示に戻す
- `setTimeout` は印刷ダイアログが閉じるまでの待機時間

## 7. 注意事項とトラブルシューティング

### 7.1 A4サイズに収まらない場合
**問題**: コンテンツ量が多く、1ページに収まらない

**対策**:
- フォントサイズを調整（各セクションで-1pt程度）
- パディング・マージンを削減（各セクション-1mm程度）
- レーダーチャートのサイズを縮小（400px → 350px）
- 軸カードのグリッドを 2列 → 4列に変更
- 質問リストを最大3項目に制限

### 7.2 レーダーチャートが印刷時に崩れる場合
**問題**: SVGチャートが印刷時に正しく表示されない

**対策**:
- Rechartsの `ResponsiveContainer` を使用せず、固定サイズを指定
- `width` と `height` を明示的に設定（例: 450px）
- 必要に応じて `viewBox` を調整

### 7.3 白黒印刷時の視認性が低い場合
**問題**: レーダーチャートの塗りつぶしが白黒で見にくい

**対策**:
- `@media print and (monochrome)` でスタイルを上書き
- 塗りつぶし色を薄いグレー（#E5E7EB）に変更
- 境界線を太く（2-3px）する
- ハッチングパターンの追加を検討（CSS `background-image`）

### 7.4 印刷ボタンが反応しない場合
**問題**: `window.print()` が動作しない、またはタイミングの問題

**対策**:
- `setTimeout` の待機時間を調整（100ms → 500ms）
- `useEffect` でDOMの準備を確認
- ブラウザのポップアップブロック設定を確認

// src/components/PrintableReport.tsx
import React from 'react';
import { RadarChart, PolarGrid, PolarAngleAxis, Radar } from 'recharts';
import './PrintableReport.css';

interface PrintableReportProps {
  scores: Array<{ name: string; score: number; description: string }>;
  respondentType: 'student' | 'parent';
  diagnosisDate: Date;
}

export default function PrintableReport({ 
  scores, 
  respondentType, 
  diagnosisDate 
}: PrintableReportProps) {
  const formatDate = (date: Date) => {
    return `${date.getFullYear()}年${date.getMonth() + 1}月${date.getDate()}日`;
  };

  return (
    <div className="printable-report">
      <header className="report-header">
        <h1>通信制高校診断結果レポート</h1>
        <div className="report-meta">
          <p>診断日: {formatDate(diagnosisDate)}</p>
          <p>回答者: {respondentType === 'student' ? '生徒' : '保護者'}</p>
        </div>
      </header>

      <section className="report-section">
        <h2>1. 診断結果の総合</h2>
        <div className="radar-container">
          <RadarChart width={450} height={450} data={scores}>
            <PolarGrid />
            <PolarAngleAxis dataKey="name" />
            <Radar 
              name="スコア" 
              dataKey="score" 
              stroke="#3B82F6" 
              fill="#3B82F6" 
              fillOpacity={0.3} 
            />
          </RadarChart>
        </div>
        <div className="axes-grid">
          {scores.map((axis, index) => (
            <div key={index} className="axis-card">
              <h3>{axis.name}</h3>

              <p className="description">{axis.description}</p>
            </div>
          ))}
        </div>
      </section>

      <footer className="report-footer">
        <p>通信制高校診断 - https://correspondence-school-finder.netlify.app</p>

  );

### B. ResultPageの修正例（handlePrint関数）

```tsx
// src/pages/ResultPage.tsx (抜粋)
const handlePrint = () => {
  // 印刷前の準備
  const printableElement = document.querySelector('.printable-report') as HTMLElement;
  if (printableElement) {
    printableElement.style.display = 'block';
  }

  // 印刷ダイアログを開く
  window.print();

  // 印刷後のクリーンアップ（印刷ダイアログが閉じた後）
  window.addEventListener('afterprint', () => {
    if (printableElement) {
      printableElement.style.display = 'none';
    }
  }, { once: true });
};
```
