### 2. セクション1: 診断結果の総合
**枠線**: 1.5px, 濃いグレー  
**高さ**: 約80mm  
**レイアウト**: 左右50%ずつの横並び配置

#### 左側: タイトルブロック（リッチデザイン）
- **装飾**:

- **配置**: 中央揃え
- **サイズ**: 180px × 180px
- **色**: 青系（#3B82F6）、塗りつぶし透過度30%

## 🎨 カラーパレット

### 主要色（カラー印刷時）
- **ヘッダー背景**: `#E5E7EB` （薄いグレー）
- **セクション番号**: `#3B82F6` （青）
- **レーダーチャート**: `#3B82F6` （青、透過度30%）
- **スコア数値**: `#3B82F6` （青）
- **ハイライト**: `#FEF3C7` （薄い黄色）
- **境界線**: `#9CA3AF` （中程度のグレー）
- **テキスト**: `#111827` （濃い黒）

### 白黒印刷時の配慮
- レーダーチャートの塗りつぶし → 薄いグレー（#D1D5DB）
- 境界線を太く（1.5px → 2px）
- テキストコントラストを強調（太字の活用）
- スコア数値に下線を追加

- **用紙サイズ**: A4 (210mm × 297mm)

### セクション高さの目安
- ヘッダー: 約10mm

- セクション間の余白: 各5mm

**合計**: 約270mm（余裕を持たせて267mm以内に収める）

## 📱 印刷時の特別な配慮

### CSS @media print による制御
1. **通常のWebページ要素を非表示**
   - ナビゲーション、フッター、ボタン類を `display: none`

2. **レポート専用レイアウトのみ表示**
   - `.printable-report` クラスを `display: block` に変更

3. **ページ分割の制御**
   - `page-break-after: avoid` で不自然な位置での改ページを防止

4. **カラー/白黒の自動対応**
   - `@media print and (monochrome)` で白黒印刷時のスタイル調整

## 🔧 実装時の技術的ポイント

### レーダーチャートの描画
- **ライブラリ**: Recharts
- **コンポーネント**: `<RadarChart>`, `<PolarGrid>`, `<Radar>`
- **固定サイズ**: `width={450}` `height={450}` （印刷時に縮小調整）

### 動的コンテンツの生成
- **質問リスト**: スコアに基づき、関連性の高い質問を自動選択

### フォントの指定
```css
font-family: 'Noto Sans JP', 'Yu Gothic', 'Meiryo', sans-serif;
```

## 💡 参考: 学習テスト結果レポートとの比較

提供いただいた参考画像（学習テスト結果レポート）と同様の構成要素:

| 要素 | 学習テストレポート | 通信制高校診断レポート |
|------|-------------------|----------------------|
| タイトル | ✅ クラス別結果レポート | ✅ 通信制高校診断結果レポート |
| 日付・受験者 | ✅ 2014年度 | ✅ 診断日・回答者区分 |
| レーダーチャート | ✅ 中央配置、8軸 | ✅ 中央配置、8軸 |
| 軸の説明 | ✅ チャート周辺にボックス | ✅ チャート下部に4×2グリッド |
| 詳細データ | ✅ グラフ（4種類） | ✅ 各軸の説明カード |
| アクション項目 | ✅ 生徒の自己評価 | ✅ 質問リスト（チェックボックス） |

**共通のデザイン思想**:
- 情報を階層的に整理
- 視覚的なチャートを中心に配置
- セクション番号で構造を明確化
- 実用性重視（チェックボックス、書き込みスペース）
