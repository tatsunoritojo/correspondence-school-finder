# プロジェクト解析・資産価値・権利関係レポート (2026-03-14)

## 1. プロジェクトの全体像と技術スタック
「こどもの進路案内所」は、通信制高校の選択を支援するWebプラットフォームであり、以下の構成で構築されています。

### コンポーネント
- **診断サイト (React + Vite + TS):** 21問の質問による8軸の適性分析。
- **ランディングページ (Next.js 14):** 集客およびSEO、アクセシビリティ対応。
- **サーバーレス (Netlify Functions):** Gemini 2.0 Flash による個別アドバイス生成。
- **PDF/画像生成 (html2canvas + jsPDF):** クライアントサイドでのレポート生成。

## 2. コードベース解析結果
- **診断ロジック (`scoring.ts`):** 8軸の平均値を算出し、ノックアウト質問で選ばれた軸に **1.4倍の重み付け** を適用。スコアは5.0でキャップ。
- **AI連携 (`gemini-advice.ts`):** 診断結果をJSON形式でAIに渡し、ユーザー属性（生徒/保護者）に応じたトーンでアドバイスを生成。
- **レポート生成 (`PrintableReport.tsx`):** 上位3軸と重要軸から、オープンスクールで確認すべき具体的質問を動的に抽出。
- **テスト:** `vitest` により計算ロジックの正確性が担保されている。

## 3. システム資産価値の試算
- **想定工数:** 約 4.3 人月（企画・設計、フロント2件、AI連携、インフラ）。
- **市場評価額:** **450万円 〜 650万円（税別）**
  - 高度な診断アルゴリズム、AIプロンプトエンジニアリング、親子データ連携の独自性を高く評価。

## 4. 権利関係と報酬に関する提言
東城氏（立ち上げメンバー・開発全担当）と永田氏（代表・発案者）の関係性において、以下の契約・合意を推奨する。

- **知的財産権:** 著作権はOne dropに帰属させる一方、東城氏の技術利用権（再利用）とクレジット表記を保証する。
- **報酬モデル:** 
  - 現在のアルバイト契約に加え、**「レベニューシェア（収益分配）」**または**「マイルストーン成功報酬」**の導入。
  - 将来の法人化を見据えた**「技術責任者(CTO)」としての役職・権限の明文化**。
- **契約形態:** 開発・保守を「業務委託」として再定義し、時給以上の「資産価値への貢献」を反映させる。
