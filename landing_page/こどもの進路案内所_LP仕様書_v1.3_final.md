## 1. プロジェクト概要

### 1.1 目的

One dropが運営する「こどもの進路案内所」のランディングページを作成する。中学校卒業後の進路選択に悩む保護者・生徒に対し、9種類の高校・進路の選択肢を紹介し、通信制高校診断サイトへの誘導を行う。

### 1.2 ターゲットユーザー

- 中学生の保護者（メイン）
- 中学生本人
- 教育関係者・支援者

### 1.3 サイト構成

- **種別**: 単一ページ（LP）のSPA
- **既存の診断サイトとの関係**: 完全に独立したページ。リンクによる外部遷移で接続
- **診断サイトリポジトリ**: `tatsunoritojo/correspondence-school-finder`（React + Vite + Tailwind CSS）

```typescript
  viewport: { once: true, margin: "-50px" },
  transition: { duration: 0.6, ease: "easeOut" }
};

export const fadeIn = {
  initial: { opacity: 0 },
  whileInView: { opacity: 1 },

};
```


### 5.3 モーダル（SchoolModal）

```typescript
// モーダルコンポーネント
function SchoolModal({ option, onClose }) {
  const [show, setShow] = useState(false);

  useEffect(() => {
    // 2フレーム後にアニメーション開始（mount直後にtransitionを発火させるため）
    requestAnimationFrame(() => requestAnimationFrame(() => setShow(true)));
    document.body.style.overflow = "hidden";
    return () => { document.body.style.overflow = ""; };
  }, []);

  const handleClose = () => {
    setShow(false);
    setTimeout(onClose, 280); // アニメーション完了後にアンマウント
  };

  // 背景: rgba(0,0,0,0) → rgba(0,0,0,0.45) fade
  // カード: opacity 0→1, scale 0.92→1, translateY 16→0
  // 閉じる: 背景タップ / ×ボタン / 「とじる」ボタン
}
```

**モーダルカードの構造**:
1. 装飾ライン（上部中央、幅40px、高さ3px）
2. 学校名（中央揃え、太字18px）
3. 区切り線
4. 説明文（両端揃え、行間2.1）
5. 「とじる」ボタン（下部中央、丸型ボーダー）

## 8. 実装上の注意事項

### 8.1 パフォーマンス

- 画像は Next.js の `<Image>` コンポーネントを使用し、自動最適化
- フォントは `next/font/google` で最適化読み込み
- Framer Motion は動的インポート（`ssr: false` は不要だが、重い場合は検討）

### 8.2 アクセシビリティ

- モーダルには `aria-label="閉じる"` を×ボタンに付与
- FAQアコーディオンには `aria-expanded` を付与
- CTAボタン・選択肢ボタンには明確な `aria-label`
- 画像には代替テキスト（`alt`）を設定
- フォントサイズは最小12px以上（モバイル）、本文は14px以上
- モーダル表示中のフォーカストラップ（推奨）

### 8.3 外部リンクの取り扱い

- 診断サイト、MAP、SNS、One drop公式サイトはすべて `target="_blank"` + `rel="noopener noreferrer"`
- 電話番号は `tel:` リンクにする（モバイルからタップで発信可能に）
